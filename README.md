# 无人机拍卖算法（UAV Auction Algorithms）

由于最近在做多无人机数字孪生的研究，其中路径规划和任务分配是除了系统开发以外两大最重要的部分，因此希望以文字记录所有的算法学习、论文阅读或者其他相关内容。

---
## 一、什么是无人机拍卖算法
- 拍卖算法是一种任务分配算法，通过竞价的方式将任务分配给无人机。分配过程中，每个无人机将提交竞价，代表的是无人机评估自身状态和任务条件的一个综合分数。然后，任务会被分配给出价最高的无人机。通过这种方式，拍卖算法可以在保证任务分配效果的同时，减少计算复杂度。

## 二、核心工作流程

1. 任务发布：一个中心控制节点或某个发起机发布任务需求，在仿真算法中，以主程序为中心控制节点。

2. 出价：每架无人机根据自己的剩余电量、当前位置、任务距离及计算能力，计算出一个“成本”或“收益”，并据此提交报价。

3. 分配任务：系统根据规则选出最优的无人机。

## 三、常见的拍卖模型
| 模型类型 | 核心特点 | 典型适用场景 |
| :--- | :--- | :--- |
| **单物品拍卖** | 一次仅处理单个任务，算法逻辑简单，响应速度极快。 | 紧急单点侦察、临时突发任务。 |
| **组合拍卖 (Combinatorial)** | 允许无人机对“一组任务”进行打包竞价，考虑任务间的关联性。 | 快递路径优化（如顺路配送多个快件）。 |
| **共识拍卖 (CBBA)** | 领域内最经典算法。通过机间通信达成一致，无需中心控制节点。 | 大规模集群协同、无基站/复杂电磁环境。 |
